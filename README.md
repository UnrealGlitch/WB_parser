Перед запуском, необходимо убедиться что установлена библиотека openpyxl.
Запускать wb_parser.py.

Если токен устарел, заменить его в файле Data/request_data.txt.
Если данные не приходят, то в файле wb_parser.py заменить dest на свое значение.
