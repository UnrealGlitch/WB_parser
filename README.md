# WB_parser
Wildberries parser

hh.ru - https://ulyanovsk.hh.ru/resume/f3eab237ff105fc7030039ed1f6f4861393876

# == Запуск ==

Перед запуском, необходимо убедиться что установлена библиотека openpyxl.
Запускать wb_parser.py.

Если токен устарел, заменить его в файле Data/request_data.txt.
Если данные не приходят, то в файле wb_parser.py заменить dest на свое значение.

*Опционально
При первом запуске можно выставить в файлу wb_parser.py значение load_from_file на False, при втором True, чтобы каждый раз не грузить с сервера одни и те же данные (при первой загрузке response будет сохранен в файл, дальше можно будет просто считывать из файла)

# == Результат ==

input.txt - содержит ответ от сервера на запрос "пальто из натуральной шерсти"
output.xlsx - содержит каталог товаров по поисковому запросу "пальто из натуральной шерсти" в XLSX формате 
Список данных:
• Ссылка на товар
• Артикул
• Название
• Цена
• Описание
• Ссылки на изображения через запятую
• Все характеристики с сохранением их структуры
• Название селлера
• Ссылка на селлера
• Размеры товара через запятую 
• Остатки по товару (число)
• Рейтинг
• Количество отзывов

sorted_output.xlsx - содержит отсортированный каталог товаров с рейтингом не менее 4.5, стоимостью до 10000 и страной производства Россия
